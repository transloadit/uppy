The `<ProgressBar />` component wraps the [ProgressBar][] plugin.

```js
import ProgressBar from 'uppy/lib/react/ProgressBar';
```

## Props

The `<ProgressBar />` component supports all [ProgressBar][] options as props.

```js
<ProgressBar
  fixed
  hideAfterFinish
/>
```
