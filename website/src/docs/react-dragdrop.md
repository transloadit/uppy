The `<DragDrop />` component wraps the [DragDrop][] plugin.

```js
import DragDrop from 'uppy/lib/react/DragDrop';
```

## Props

The `<DragDrop />` component supports all [DragDrop][] options as props.

```js
<DragDrop
  width="100%"
  height="100%"
  note="Images up to 200Ã—200px"
/>
```
